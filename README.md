# Game-Assistant
该系列脚本-游戏助手，是一系列用于游戏脚本开发和游戏代肝的脚本。目前该系列有MAK，SAG，分别适用于环行旅舍和原神。  
function.py内有依赖于绿色版tessdata的ocr功能函数，如需使用OCR功能，请在蓝奏云/百度云下载tessdata模块压缩包，置于主目录下。若无需使用则可以不添加。
